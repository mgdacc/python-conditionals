Estructura: if

Ejecuta un bloque de código si la condición es verdadera.

edad = 20
if edad >= 18:
    print("Eres mayor de edad.")
2 / 18

Estructura: elif

Si el if es falso, comprueba esta otra condición.

nota = 85
if nota >= 90:
    print("Sobresaliente")
elif nota >= 80:
    print("Notable")
3 / 18

Estructura: else

Se ejecuta si ninguna condición anterior fue verdadera.

if nota >= 50:
    print("Aprobado")
else:
    print("Suspendido")

Ejemplo Completo

Combinando `if`, `elif` y `else`.

edad = 15
if edad < 13:
    print("Niño")
elif edad < 18:
    print("Adolescente")
else:
    print("Adulto")

Lógica: and

Verdadero solo si AMBAS condiciones son verdaderas.

edad = 25
if edad > 18 and edad < 30:
    print("Estás en tus 20s.")

Lógica: or

Verdadero si AL MENOS UNA condición es verdadera.

dia = "Sábado"
if dia == "Sábado" or dia == "Domingo":
    print("Es fin de semana.")

Lógica: not

Invierte el resultado de una condición.

if not es_festivo:
    print("A trabajar.")

Valores "Falsy"

Valores que se evalúan como Falso en un `if`.

# Estos son Falsy:
False, None, 0, "", [], {}, ()
15 / 18

Ejemplo con "Falsy"

Un `if` sobre una lista vacía es Falso.

lista_compras = []
if lista_compras:
    print("Hay que comprar.")
else:
    print("La lista está vacía.")

Valores "Truthy"

Cualquier valor que no sea "Falsy" se evalúa como Verdadero.

nombre = "Ana"
if nombre: # Esto es True
    print(f"Hola, {nombre}")

Operador Ternario

Una forma concisa de escribir un `if-else` simple.

edad = 20
mensaje = "Mayor" if edad >= 18 else "Menor"
print(mensaje) # "Mayor"
